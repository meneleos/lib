Thank you for downloading our MatriX XMPP SDK.

MatriX is a SDK for the extensible Messaging and Presence Protocol (XMPP) protocol written in c#. This SDK can be used to build high quality XMPP software applications.

Examples:

In the samples directory you can find a chat client. This example is a good tutorial to get started with MatriX and XMPP development. We will add more examples and tutorials with upcoming releases.

Because the API and usage is exactly the same with all MatriX editions we suggest to download also the other editions which include other examples.

Documentation:

  • API Documentation is included with all releases in CHM format.
  • MatriX developer online guide
  • To learn more about XMPP, get familiar with XMPP concepts  and real time communications we suggest the following book:
    XMPP - The Definitive Guide
